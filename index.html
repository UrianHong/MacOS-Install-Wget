在MacOS系统上安装Weget

如何在MacOS上安装wget

Download this project as a .zip file Download this project as a tar.gz file

下载一个Wget的源码包，http://www.gnu.org/software/wget/
打开终端解压
1、输入：tar zxvf wget-1.9.1.tar.gz
2、cd 进入到解压的目录
3、输入：./configure
4、输入：make
5、输入：sudo make install
安装过程中make会提示出错。
checking for compress in -lz... yes
checking for gpg_err_init in -lgpg-error... no
checking for gcry_control in -lgcrypt... no
checking for libgnutls... no
configure: error: --with-ssl was given, but GNUTLS is not available.
解决方法也很简单：
# tar zxvf wget-1.13.4.tar.gz
# cd wget-1.13.4
wget-1.13.4 root# ./configure --with-ssl=openssl
wget-1.13.4 root# make
wget-1.13.4 root# make install
注意，你如果不是使用root用户，那么在安装的这一步需要使用sudo授权。
当然有更简单的方法，那就是用MAC的软件管理包来安装咯！
mac下面的package manager，自动download、compile、install，并且会自动寻找依赖关系的包然后自动安装的。比较好的软件包管理器有：
homebrew （http://mxcl.github.com/homebrew/）推荐，专门来替代MacPorts
MacPorts (http://www.macports.org/)
刚刚试过，这两个软件包管理器，都是有 wget 的安装源的。这里是搜索结果：
mac:~ juven$ brew search wget
wget
mac:~ juven$ port search wget
wget @1.13.4 (net, www)
internet file retriever
安装完毕 homebrew或者MacPorts后，直接
brew install wget
或者
port install wget
